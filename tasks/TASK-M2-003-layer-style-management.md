# TASK-M2-003：完整文字样式与图层管理

**状态**：已完成（2026-07-16）  
**依赖**：TASK-M2-002

实现字体、字号、颜色、对齐、换行、旋转、描边、阴影以及新增/删除文字框；每项操作进入统一命令历史，并验证字体缺字、透明通道和溢出状态。

## 完成记录

- 正式 `TextStyle` 支持字体、手动字号/自动适配、填充色、水平/垂直对齐、换行、描边颜色/宽度、阴影颜色/透明度/X-Y 偏移；旋转继续由 `TextBox` 持有。
- Qt 渲染保持完整字符串 shaping；描边使用同一布局的轮廓偏移合成，阴影使用同一布局的透明偏移合成，不拆分复杂脚本字符。
- 新增文字框默认位于图片中心并生成稳定会话 ID；删除、新增、样式修改和既有文字/几何修改共用有界命令历史。
- 用户可以在“样式”页选择已安装字体、编辑全部基础样式并新增/删除框；渲染成功后才提交命令。
- 自动识别原字体、字体相似度映射、艺术字和弧形字不在本任务内，继续由 TASK-M2-007 实现。
